Pebble 301 Redirects is a simple tool to manage redirects on your Wordpress site.

== Description ==

Pebble 301 Redirects provides an incredibly simple dashboard to manage your 301 redirects to make sure your visitors land on the page they were expecting. When creating new pages or upgrading your website, new pages are often created - use Pebble 301 Redirects to not only redirect your users to the new page, but carry across your SEO juice from the original page.

== Installation ==

1. Upload Pebble 301 Redirects to the your plugins directory (wp-content/plugins)
1. Activate the plugin (Admin dashboard -> Plugins)
1. Visit Settings -> 301 Redirects to start adding your redirects
